# Views Folder

This folder contains all HTML view files for the Spendy application.

## Structure

All HTML pages are located here, including:
- Landing pages (index.html is in root, but other landing pages are here)
- Dashboard pages (DashboardExpenses.html, DashboardIncome.html)
- Settings and profile pages
- Authentication pages (login.html, SignUp.html)

## Navigation

HTML files in this folder can reference each other using relative paths (e.g., `savings.html`, `profile.html`).

## API Calls

All API calls should use the path: `../api/filename.php`

## Assets

- Images: `../images/`
- JavaScript: `../js/`
- Other assets: `../images/` (for logos, backgrounds, etc.)
